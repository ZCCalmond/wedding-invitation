## 🎨 如何调整地图参数

### 调整缩放级别（zoom）

在 `mapNavigation.js` 中修改 `zoom` 参数：

```javascript
// zoom=12  - 城市级别（看到整个城市）
// zoom=14  - 区域级别（看到几条街道）
// zoom=16  - 街道级别（推荐，看清周边）
// zoom=18  - 建筑级别（最大，看到建筑细节）

// 当前使用
mapPlaceholder: `https://apis.map.qq.com/ws/staticmap/v2/?center=33.783994,119.80249&zoom=16&...`

// 如果想看得更广一点
mapPlaceholder: `https://apis.map.qq.com/ws/staticmap/v2/?center=33.783994,119.80249&zoom=14&...`
```

### 调整地图尺寸

```javascript
// 当前尺寸：800*400
// 可以修改为：
&size=1000*500   // 更大
&size=600*300    // 更小
```

### 修改标记样式

```javascript
// 当前：红色标记，标签为 A
&markers=color:red|label:A|33.783994,119.80249

// 可以修改为：
&markers=color:blue|label:婚礼|33.783994,119.80249  // 蓝色，标签为"婚礼"
&markers=color:green|label:❤️|33.783994,119.80249   // 绿色，标签为爱心
```

## 🔧 常见问题

### Q: 地图显示不出来怎么办？
A: 
1. 检查网络连接
2. 确认腾讯地图 API 服务正常
3. 查看浏览器控制台是否有错误

### Q: 标记位置不准确？
A: 
1. 检查坐标是否正确
2. 腾讯地图使用的坐标系是 GCJ-02
3. 可以在腾讯地图网页版查找准确坐标：https://lbs.qq.com/getPoint/

### Q: 想要更好的效果？
A: 可以升级到嵌入式互动地图（方案 4），用户可以缩放、拖动。

### Q: API Key 会过期吗？
A: 这是测试 Key，如果长期使用建议申请自己的 Key：
1. 注册腾讯位置服务：https://lbs.qq.com/
2. 创建应用
3. 获取 Key
4. 替换配置中的 Key

## 📊 地图参数完整说明

```javascript
https://apis.map.qq.com/ws/staticmap/v2/
  ?center=33.783994,119.80249    // 地图中心点（纬度,经度）
  &zoom=16                        // 缩放级别（3-18）
  &size=800*400                   // 图片尺寸（宽*高）
  &markers=                       // 标记配置
    color:red|                    // 颜色（red/blue/green等）
    label:A|                      // 标签文字
    33.783994,119.80249          // 标记坐标
  &key=YOUR_KEY                   // API Key
```
